# FCCalc - Калькулятор для електротехнічних розрахунків

Android додаток для виконання електротехнічних розрахунків, розроблений на Kotlin з використанням Jetpack Compose.

## Функціонал

Додаток містить три модулі розрахунків:

### 1. Вибір кабелів
- Розрахунок оптимального перерізу кабелів для електропостачання
- Врахування економічної густини струму
- Перевірка термічної стійкості до струмів короткого замикання
- Рекомендації щодо типу та перерізу кабелю

### 2. Струми КЗ на шинах ГПП
- Розрахунок струмів короткого замикання на шинах головних понижувальних підстанцій
- Визначення параметрів системи електропостачання

### 3. Струми КЗ підстанції
- Розрахунок струмів короткого замикання для трансформаторних підстанцій
- Аналіз режимів короткого замикання

## Технології

- **Мова**: Kotlin 2.0.21
- **UI**: Jetpack Compose + Material3
- **Архітектура**: MVVM з StateFlow
- **Навігація**: Navigation Compose
- **Min SDK**: Android 10 (API 29)
- **Target SDK**: Android 14 (API 36)

## Встановлення та запуск

### Передумови
- Java JDK 11 або вище
- Android SDK
- Android Studio (рекомендовано)

### Збірка проекту

Перейдіть до директорії `FCCalc/`:

```bash
cd FCCalc
```

**Debug збірка:**
```bash
./gradlew assembleDebug
```

APK буде створено в: `app/build/outputs/apk/debug/app-debug.apk`

**Release збірка:**
```bash
./gradlew assembleRelease
```

**Для Windows також доступні:**
- `build.bat` - швидка збірка
- `install_and_run.bat` - збірка та встановлення на пристрій

### Встановлення на пристрій

```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

## Структура проекту

```
FCCalc/
├── app/src/main/java/com/fccalc/
│   ├── data/models/         # Моделі даних для розрахунків
│   ├── ui/
│   │   ├── screens/        # Екрани додатку
│   │   ├── viewmodels/     # ViewModels з бізнес-логікою

│   │   └── theme/          # Тема оформлення
│   ├── navigation/         # Налаштування навігації
│   └── MainActivity.kt     # Точка входу
└── BUILD.md                # Детальна інструкція зі збірки
```
